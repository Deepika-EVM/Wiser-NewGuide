Image Settings

üñºÔ∏è Image Aspect Ratios settings

  • Square (1:1): square shape, Portrait (4:5): vertical image, Tall (2:3): tall vertical image, Short (4:3): wide horizontal image, Square (1:1) ‚Äì fill card: square image fills card, Portrait (4:5) ‚Äì fill card: vertical image fills card, Tall (2:3) ‚Äì fill card: taller image fills card height, Short (4:3) ‚Äì fill card: horizontal image fills width.

üñºÔ∏è Image Settings ‚Äì Overview

  • Square (1:1): A square image has equal width and height, making it perfect for symmetrical product photos or icons. It keeps everything balanced and visually even.
    **Example:** Product images that need to be perfectly proportioned for a clean, simple look.
  • Portrait (4:5): A portrait image is slightly taller than it is wide. This is best for vertical images like fashion photos, art, or portraits, where the focus is on the height of the subject.
    **Example:** Fashion photos, artwork, or close-up portraits.
  • Tall (2:3): A tall image is even more vertical than the portrait ratio, offering a long vertical shape. It's great for editorial images, tall product shots, or portrait-style photos where height is the focus.
    **Example:** Vertical editorial shots, or photos where the subject is taller than wide.
  • Short (4:3): A short image is wider than it is tall. This is ideal for landscape shots or wide-angle images, as it provides more width than height.
    **Example:** Landscape photography, group shots, or wide-angle views of products or scenes.

üß© Fill Card Variants

  • Square (1:1) ‚Äì fill card: A square image will completely fill the card, keeping the width and height equal, so there are no gaps or empty spaces within the card.
    **Usage:** Ideal for products that need equal width and height, such as small square-shaped items or icons.
  • Portrait (4:5) ‚Äì fill card: A portrait image will stretch vertically to fill the card. It‚Äôs taller, but it may lose some width depending on the card‚Äôs dimensions.
    **Usage:** Best for vertical images like clothing or artwork that need to take up the full height of the card while adjusting slightly in width.
  • Tall (2:3) ‚Äì fill card: A tall image will fill the height of the card. If the card is taller than the image, the width will be adjusted accordingly.
    **Usage:** Perfect for images with more height, such as tall product shots, posters, or editorial images.
  • Short (4:3) ‚Äì fill card: A short image will stretch horizontally to fill the card‚Äôs width. The height will be adjusted proportionally, ensuring there are no empty spaces.
    **Usage:** Ideal for wide images like landscapes, wide-angle product shots, or group photos.

  • Button Type: Decide how the button should behave:
    • button ‚Äì Performs actions like "Add to Cart"
    • a ‚Äì Navigates to a link or product page
  • Theme Support: If you use class names from your Shopify theme (e.g., btn, btn--primary), the theme's built-in CSS will style the button automatically ‚Äî no need to write custom CSS unless needed.

Badge Settings

  • Add Class Name: In the input box below, you can enter a CSS class name for your badge (e.g., badge-red, badge-outline, badge-glow). If your theme already includes styling for that class, you don‚Äôt need to write any extra CSS.

  • Product Tags Badge: Displays a custom badge when a product has specific tags. For example, if a product tag is toys, you can show a badge like 30 off at here.
  • Save Badge: Shows discount badges based on product pricing. You can display percentage off (e.g., 20% off) or amount saved (e.g., $5 save) based on your settings.
  • Product Type Badge: Automatically displays a badge showing the product type, like Shirt or Accessories, based on the product‚Äôs type.
  • Add JSON in Max Badge: You can paste your badge configuration JSON in the box above to customize which badges appear based on tag, type, or pricing rules.

üè∑Ô∏è Badge Settings ‚Äì Overview

  • Add Class Name: In the input box below, you can enter a custom CSS class for your badge, such as badge-red, badge-outline, or badge-glow. If your theme already supports that class, it will automatically apply the styles‚Äîno extra CSS needed.

The product type (such as "New Arrival", "Sale", "Featured", etc.) will be shown dynamically as the badge on your product card.

  }

You can choose between showing the discount as a percentage (e.g., 20% off) by setting "percentage": true, or as an amount saved (e.g., $10 save) by using "savePrice": true. Only one of these will display at a time, and you can control which one gets priority using "percentage_text" and "savePrice_text" to customize the ending text‚Äîlike "off" or "saved".

Pagination Autplay Navigation Scrollbar Slider Behavior

Pagination

  • Enable Pagination: Turn this setting on to display pagination controls (like dots or numbers) below your slider. This helps users know which slide they are viewing.
  • Clickable: When enabled, users can click the pagination bullets to jump directly to a specific slide. If disabled, the bullets are only for display.
  • Dynamic Bullets: This option shows a smaller set of bullets centered around the active slide, which is helpful when there are many slides. It makes the pagination look cleaner.
  • Pagination Type Dropdown: Choose the style of pagination from this dropdown. Options usually include: bullets (small dots), fraction (e.g., 2/5), progressbar (a sliding bar), or none

Autoplay

Enable Autoplay: Turn this setting on to make the slider automatically switch slides without user interaction.

Autoplay Time: Set the delay between each slide change. For example, 2000 means a 2-second delay between slides.

Navigation

  • Enable Navigation: Turn this setting on to show previous/next arrows for manually navigating through the slides.
  • Arrow Width (in px): Set the width of the navigation arrows. Example: 37px for a standard arrow size.
  • Arrow Height (in px): Set the height of the navigation arrows to match your design. Example: 37px.
  • Arrow Border Radius (in %): Adjust the corner roundness of the arrows. Example: 30% gives a soft rounded arrow button.
  • Add Custom Arrow HTML/SVG: You can paste your own HTML or SVG code to fully customize the appearance of the arrows. This is useful if your theme or brand has unique arrow styles.

Scrollbar

  • Enable Scrollbar: Turn this setting on to display a horizontal scrollbar below the slider, allowing manual slide navigation.
  • Show on Hover: Enable this option to make the scrollbar visible only when the user hovers over the slider area. This helps keep the UI clean while still offering interactive control.

Slider Behavior

  • Enable Direction (Enable Dir): Choose the scroll direction: ltr (left-to-right) or rtl (right-to-left) depending on layout.
  • Loop: Makes the slider repeat endlessly. After reaching the last slide, it loops back to the first.
  • Centered Slides: Keeps the active slide in the center. This is ideal for partial slides on the edges.
  • CSS Mode: Enables native browser scrolling with snap behavior. Useful for simple, fast sliders.
  • Resize Observer: Automatically adjusts the slider when its container or content changes size.
  • Space Between (in px): Defines the horizontal gap between slider cards. For example, 10 adds a 10px space between each card.
  • Add Text: This field can also be used to insert custom JavaScript (like Swiper.js settings) for responsive behavior. For example, you can control how many slides are visible based on screen width using slidesPerView with fractional values. This makes part of the next slide visible, improving the visual flow.

    '{'loop':false,'breakpoints':{'0':{'slidesPerView':1,'spaceBetween':10},'300':{'slidesPerView':1.1,'spaceBetween':15},'768':{'slidesPerView':1.5,'spaceBetween':15},
